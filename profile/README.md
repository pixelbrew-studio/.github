# Pixelbrew Studio

Independent AI-native product lab for small tools, public experiments, and original software.

Pixelbrew is not an agency or consultancy. It is a public workbench for tools that are narrow enough to understand, useful enough to reuse, and opinionated enough to carry a point of view.

## What We Publish

- Small TypeScript packages for evidence, evaluation, and product engineering.
- Local command-line tools that make AI-assisted development more disciplined.
- Clean-room experiments that can be reused as code, patterns, or references.

## Public Repos

- [pb-suite](https://github.com/pixelbrew-studio/pb-suite) — local command suite for reviews, QA, copy checks, security passes, and release discipline.
- [quote-locator](https://github.com/pixelbrew-studio/quote-locator) — TypeScript utility for locating claimed quotes inside source text.
- [eval-metrics-ts](https://github.com/pixelbrew-studio/eval-metrics-ts) — dependency-light metrics for classification and ranking evaluations.

## Flagship

[Evalgist](https://evalgist.ai) is the current flagship: evidence-grounded AI evaluation software for privacy-sensitive document workflows.

Evalgist product code is company-owned and private. Pixelbrew publishes clean-room tools and patterns around the craft of building careful AI-native software without publishing Evalgist implementation details, prompts, fixtures, screenshots, audits, or internal findings.
