# Pixelbrew Studio

Independent AI-native product lab for small, useful tools, experiments, and original software.

Pixelbrew is not an agency or consultancy. It is a public workbench for tools that are small enough to understand, useful enough to reuse, and opinionated enough to have a point of view.

## Public Work

- [pb-suite](https://github.com/pixelbrew-studio/pb-suite) — local command suite for disciplined AI-assisted product engineering.
- [quote-locator](https://github.com/pixelbrew-studio/quote-locator) — locate claimed quotes inside source text.
- [eval-metrics-ts](https://github.com/pixelbrew-studio/eval-metrics-ts) — small evaluation metrics for TypeScript projects.

## Flagship

[Evalgist](https://evalgist.ai) is the current flagship: evidence-grounded AI evaluation software for privacy-sensitive document workflows. Evalgist product code is company-owned and private; Pixelbrew publishes clean-room tools, patterns, and experiments around the craft of building such systems.
